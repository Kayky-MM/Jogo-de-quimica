Como jogar?

No seu turno, uma pergunta aparecerá na tela. Você terá alguns segundos para respondê-la. Se você errar, a vez passa para o próximo jogador. Se acertar, você pode clicar no dado para descobrir quantas casas pode avançar no tabuleiro.

Como Responder?

Cada pergunta virá em um "card" que mostra o enunciado, um exemplo de como deve ser a resposta e o campo para você digitar. Preste muita atenção nestes detalhes:

  • Para respostas de uma ou duas palavras, a ortografia correta é fundamental.
  • Se a resposta incluir uma unidade de medida (como "g/L", "mol", "mol/L"), certifique-se de deixar um espaço entre o número e a unidade (exemplo: "2 mol/L", não "2mol/L").
  • Em caso de qualquer dúvida sobre o formato da resposta, fale imediatamente com o organizador(a) do jogo. Ele(a) poderá te ajudar a preencher o campo corretamente.

Casas Especiais

Se você cair em uma casa escura, o jogo ativará um dos quatro efeitos especiais. Fique atento e tente memorizar o que cada casa faz, pois outros jogadores (e você novamente) podem cair nelas no futuro!

  • Avance uma casa

    Você move seu peão uma casa para frente.

  • Volte uma casa

    Você move seu peão uma casa para trás.

  • Volte ao Início

    Você retorna para a casa de partida (Casa 1).

  • Passe a Vez

    Você não jogará na próxima rodada, e a vez passará direto para o próximo jogador.
